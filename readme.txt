1. I Arduino editorn gå in i file > Examples > Firmata > Standard Firmata och ladda upp koden till arduinon
2. Ladda ner python 3.4 eller senare, lägg in i PATH
3. i terminalen skriv "pip install pyfirmata", detta borde ladda ner library:t
4. Ändra COM3 till rätt port i car_controls2.py, är inte helt säker på hur det ska skrivas ännu, man kanske måste hitta rätt path.
5. kör "python car_controls2.py" i terminalen
